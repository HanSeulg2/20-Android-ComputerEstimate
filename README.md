# Android/컴퓨터조립견적 Program
- Language : Android
- Tools : Android Studio 4.1.1
- Techs : Kakao api, Naver api
- Database : Firebase
- 형성관리 : Smartgit
### 주요기능
- 카카오 로그인 및 마이페이지
- 부품 조회, 선택
- 유저 채팅
